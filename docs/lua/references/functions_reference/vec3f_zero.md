
## [vec3f_zero](#vec3f_zero)

### Description
Sets the components of the 3D floating-point vector `v` to 0

### Lua Example
`local vec3fValue = vec3f_zero(v)`

### Parameters
| Field | Type |
| ----- | ---- |
| v | [Vec3f](structs.md#Vec3f) |

### Returns
- [Vec3f](structs.md#Vec3f)

### C Prototype
`Vec3fp vec3f_zero(VEC_OUT Vec3f v);`
