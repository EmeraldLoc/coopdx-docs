
## [get_lighting_dir](#get_lighting_dir)

### Description
Gets a value of the global lighting direction

### Lua Example
`local numberValue = get_lighting_dir(index)`

### Parameters
| Field | Type |
| ----- | ---- |
| index | `integer` |

### Returns
- `number`

### C Prototype
`f32 get_lighting_dir(u8 index);`
