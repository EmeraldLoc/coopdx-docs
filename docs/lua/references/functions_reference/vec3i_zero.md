
## [vec3i_zero](#vec3i_zero)

### Description
Sets the components of the 3D integer vector `v` to 0

### Lua Example
`local vec3iValue = vec3i_zero(v)`

### Parameters
| Field | Type |
| ----- | ---- |
| v | [Vec3i](structs.md#Vec3i) |

### Returns
- [Vec3i](structs.md#Vec3i)

### C Prototype
`Vec3ip vec3i_zero(VEC_OUT Vec3i v);`
