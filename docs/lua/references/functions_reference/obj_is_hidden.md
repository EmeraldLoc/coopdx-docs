
## [obj_is_hidden](#obj_is_hidden)

### Description
Returns `TRUE` if the given object is currently hidden from rendering

### Lua Example
`local integerValue = obj_is_hidden(obj)`

### Parameters
| Field | Type |
| ----- | ---- |
| obj | [Object](structs.md#Object) |

### Returns
- `integer`

### C Prototype
`s32 obj_is_hidden(struct Object *obj);`
