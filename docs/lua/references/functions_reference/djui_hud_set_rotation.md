
## [djui_hud_set_rotation](#djui_hud_set_rotation)

### Description
Sets the current DJUI HUD rotation

### Lua Example
`djui_hud_set_rotation(rotation, pivotX, pivotY)`

### Parameters
| Field | Type |
| ----- | ---- |
| rotation | `integer` |
| pivotX | `number` |
| pivotY | `number` |

### Returns
- None

### C Prototype
`void djui_hud_set_rotation(s16 rotation, f32 pivotX, f32 pivotY);`
