
## [set_dialog_box_state](#set_dialog_box_state)

### Description

### Lua Example
`set_dialog_box_state(state)`

### Parameters
| Field | Type |
| ----- | ---- |
| state | `integer` |

### Returns
- None

### C Prototype
`void set_dialog_box_state(u8 state);`
