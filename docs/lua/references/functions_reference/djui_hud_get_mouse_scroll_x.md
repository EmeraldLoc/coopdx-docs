
## [djui_hud_get_mouse_scroll_x](#djui_hud_get_mouse_scroll_x)

### Description
Returns the amount scrolled horizontally (-left/right+)

### Lua Example
`local numberValue = djui_hud_get_mouse_scroll_x()`

### Parameters
- None

### Returns
- `number`

### C Prototype
`f32 djui_hud_get_mouse_scroll_x(void);`
