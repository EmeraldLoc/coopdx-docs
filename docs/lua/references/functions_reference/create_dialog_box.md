
## [create_dialog_box](#create_dialog_box)

### Description
Creates a dialog box with a dialog ID that rotates into view

### Lua Example
`create_dialog_box(dialog)`

### Parameters
| Field | Type |
| ----- | ---- |
| dialog | `integer` |

### Returns
- None

### C Prototype
`void create_dialog_box(s32 dialog);`
