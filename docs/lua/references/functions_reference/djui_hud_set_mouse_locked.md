
## [djui_hud_set_mouse_locked](#djui_hud_set_mouse_locked)

### Description
Locks (or unlocks) the cursor to the window

### Lua Example
`djui_hud_set_mouse_locked(locked)`

### Parameters
| Field | Type |
| ----- | ---- |
| locked | `boolean` |

### Returns
- None

### C Prototype
`void djui_hud_set_mouse_locked(bool locked);`
