
## [fadeout_background_music](#fadeout_background_music)

### Description
Fades out background music `seqId` over `fadeOut`

### Lua Example
`fadeout_background_music(seqId, fadeOut)`

### Parameters
| Field | Type |
| ----- | ---- |
| seqId | `integer` |
| fadeOut | `integer` |

### Returns
- None

### C Prototype
`void fadeout_background_music(u16 seqId, u16 fadeOut);`
