
## [sound_get_level_intensity](#sound_get_level_intensity)

### Description
Gets a sound level intensity based on `distance`

### Lua Example
`local numberValue = sound_get_level_intensity(distance)`

### Parameters
| Field | Type |
| ----- | ---- |
| distance | `number` |

### Returns
- `number`

### C Prototype
`f32 sound_get_level_intensity(f32 distance);`
