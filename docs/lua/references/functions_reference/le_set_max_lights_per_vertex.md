
## [le_set_max_lights_per_vertex](#le_set_max_lights_per_vertex)

### Description
Sets the max amount of lights that can affect a vertex

### Lua Example
`le_set_max_lights_per_vertex(count)`

### Parameters
| Field | Type |
| ----- | ---- |
| count | `integer` |

### Returns
- None

### C Prototype
`void le_set_max_lights_per_vertex(u8 count);`
