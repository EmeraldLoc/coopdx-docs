
## [reset_rumble_timers](#reset_rumble_timers)

### Description
Resets rumble timers only if `m` is the local Mario

### Lua Example
`reset_rumble_timers(m)`

### Parameters
| Field | Type |
| ----- | ---- |
| m | [MarioState](structs.md#MarioState) |

### Returns
- None

### C Prototype
`void reset_rumble_timers(struct MarioState* m);`
