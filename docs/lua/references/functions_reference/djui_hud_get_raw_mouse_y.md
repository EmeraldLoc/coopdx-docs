
## [djui_hud_get_raw_mouse_y](#djui_hud_get_raw_mouse_y)

### Description
Returns the y coordinate of the mouse relative to the screen

### Lua Example
`local numberValue = djui_hud_get_raw_mouse_y()`

### Parameters
- None

### Returns
- `number`

### C Prototype
`f32 djui_hud_get_raw_mouse_y(void);`
