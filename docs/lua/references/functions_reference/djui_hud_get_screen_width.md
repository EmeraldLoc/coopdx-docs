
## [djui_hud_get_screen_width](#djui_hud_get_screen_width)

### Description
Gets the screen width in the current DJUI HUD resolution

### Lua Example
`local integerValue = djui_hud_get_screen_width()`

### Parameters
- None

### Returns
- `integer`

### C Prototype
`u32 djui_hud_get_screen_width(void);`
