
## [set_fog_intensity](#set_fog_intensity)

### Description
Sets the intensity of the fog (this value scales very quickly, 1.0 to 1.1 is a desirable range)

### Lua Example
`set_fog_intensity(intensity)`

### Parameters
| Field | Type |
| ----- | ---- |
| intensity | `number` |

### Returns
- None

### C Prototype
`void set_fog_intensity(f32 intensity);`
