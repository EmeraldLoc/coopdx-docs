
## [set_sound_moving_speed](#set_sound_moving_speed)

### Description
Sets the `speed` of moving `bank`

### Lua Example
`set_sound_moving_speed(bank, speed)`

### Parameters
| Field | Type |
| ----- | ---- |
| bank | `integer` |
| speed | `integer` |

### Returns
- None

### C Prototype
`void set_sound_moving_speed(u8 bank, u8 speed);`
