
## [get_sound_pan](#get_sound_pan)

### Description
Gets a sound left/right pan using `x` and `z`

### Lua Example
`local numberValue = get_sound_pan(x, z)`

### Parameters
| Field | Type |
| ----- | ---- |
| x | `number` |
| z | `number` |

### Returns
- `number`

### C Prototype
`f32 get_sound_pan(f32 x, f32 z);`
