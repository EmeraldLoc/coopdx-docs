
## [cur_obj_has_model](#cur_obj_has_model)

### Description
Checks whether the current object uses the specified model geometry

### Lua Example
`local integerValue = cur_obj_has_model(modelID)`

### Parameters
| Field | Type |
| ----- | ---- |
| modelID | `integer` |

### Returns
- `integer`

### C Prototype
`s32 cur_obj_has_model(u16 modelID);`
