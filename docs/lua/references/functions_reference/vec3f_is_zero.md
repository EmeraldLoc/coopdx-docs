
## [vec3f_is_zero](#vec3f_is_zero)

### Description
Returns `true` if all components of the 3D floating-point vector `v` are zero

### Lua Example
`local booleanValue = vec3f_is_zero(v)`

### Parameters
| Field | Type |
| ----- | ---- |
| v | [Vec3f](structs.md#Vec3f) |

### Returns
- `boolean`

### C Prototype
`bool vec3f_is_zero(Vec3f v);`
