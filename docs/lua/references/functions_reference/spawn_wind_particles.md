
## [spawn_wind_particles](#spawn_wind_particles)

### Description
Spawns wind particles around the current object

### Lua Example
`spawn_wind_particles(pitch, yaw)`

### Parameters
| Field | Type |
| ----- | ---- |
| pitch | `integer` |
| yaw | `integer` |

### Returns
- None

### C Prototype
`void spawn_wind_particles(s16 pitch, s16 yaw);`
