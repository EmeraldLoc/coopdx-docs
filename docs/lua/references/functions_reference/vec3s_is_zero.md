
## [vec3s_is_zero](#vec3s_is_zero)

### Description
Returns `true` if all components of the 3D short integer vector `v` are zero

### Lua Example
`local booleanValue = vec3s_is_zero(v)`

### Parameters
| Field | Type |
| ----- | ---- |
| v | [Vec3s](structs.md#Vec3s) |

### Returns
- `boolean`

### C Prototype
`bool vec3s_is_zero(Vec3s v);`
