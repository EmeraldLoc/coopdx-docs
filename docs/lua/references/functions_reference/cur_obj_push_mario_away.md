
## [cur_obj_push_mario_away](#cur_obj_push_mario_away)

### Description
Pushes any player within a radius away from the current object on the XZ plane

### Lua Example
`cur_obj_push_mario_away(radius)`

### Parameters
| Field | Type |
| ----- | ---- |
| radius | `number` |

### Returns
- None

### C Prototype
`void cur_obj_push_mario_away(f32 radius);`
