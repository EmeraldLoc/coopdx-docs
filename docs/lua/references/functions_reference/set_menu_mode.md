
## [set_menu_mode](#set_menu_mode)

### Description
Sets the in-game menu state. 0-1 is the courses box with the castle secret stars and 2-3 is the course completion screen.

### Lua Example
`set_menu_mode(mode)`

### Parameters
| Field | Type |
| ----- | ---- |
| mode | `integer` |

### Returns
- None

### C Prototype
`void set_menu_mode(s16 mode);`
