
## [set_dialog_override_pos](#set_dialog_override_pos)

### Description
Dialog box customization: Sets the override position for a dialog box

### Lua Example
`set_dialog_override_pos(x, y)`

### Parameters
| Field | Type |
| ----- | ---- |
| x | `integer` |
| y | `integer` |

### Returns
- None

### C Prototype
`void set_dialog_override_pos(s16 x, s16 y);`
