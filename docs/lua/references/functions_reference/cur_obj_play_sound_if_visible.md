
## [cur_obj_play_sound_if_visible](#cur_obj_play_sound_if_visible)

### Description
Plays a sound if the current object is visible

### Lua Example
`cur_obj_play_sound_if_visible(soundMagic)`

### Parameters
| Field | Type |
| ----- | ---- |
| soundMagic | `integer` |

### Returns
- None

### C Prototype
`void cur_obj_play_sound_if_visible(s32 soundMagic);`
