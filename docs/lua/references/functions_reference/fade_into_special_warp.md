
## [fade_into_special_warp](#fade_into_special_warp)

### Description
Fades into a special warp with `arg` and using `color`

### Lua Example
`fade_into_special_warp(arg, color)`

### Parameters
| Field | Type |
| ----- | ---- |
| arg | `integer` |
| color | `integer` |

### Returns
- None

### C Prototype
`void fade_into_special_warp(u32 arg, u32 color);`
