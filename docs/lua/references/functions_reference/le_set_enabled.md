
## [le_set_enabled](#le_set_enabled)

### Description
This will let the user control the lighting engine in real time to disable or enable it.

### Lua Example
`le_set_enabled(value)`

### Parameters
| Field | Type |
| ----- | ---- |
| value | `boolean` |

### Returns
- None

### C Prototype
`void le_set_enabled(bool value);`
