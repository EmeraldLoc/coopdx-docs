
## [vec3s_zero](#vec3s_zero)

### Description
Sets the components of the 3D short integer vector `v` to 0

### Lua Example
`local vec3sValue = vec3s_zero(v)`

### Parameters
| Field | Type |
| ----- | ---- |
| v | [Vec3s](structs.md#Vec3s) |

### Returns
- [Vec3s](structs.md#Vec3s)

### C Prototype
`Vec3sp vec3s_zero(VEC_OUT Vec3s v);`
