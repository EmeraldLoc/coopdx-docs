
## [le_get_light_radius](#le_get_light_radius)

### Description
Gets a lighting engine point light's `radius`

### Lua Example
`local numberValue = le_get_light_radius(id)`

### Parameters
| Field | Type |
| ----- | ---- |
| id | `integer` |

### Returns
- `number`

### C Prototype
`f32 le_get_light_radius(s16 id);`
