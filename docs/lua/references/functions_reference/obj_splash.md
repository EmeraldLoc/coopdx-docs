
## [obj_splash](#obj_splash)

### Description
Generates splashes if at surface of water, entering water, or bubbles if underwater

### Lua Example
`obj_splash(waterY, objY)`

### Parameters
| Field | Type |
| ----- | ---- |
| waterY | `integer` |
| objY | `integer` |

### Returns
- None

### C Prototype
`void obj_splash(s32 waterY, s32 objY);`
