
## [vec3s_dot](#vec3s_dot)

### Description
Computes the dot product of the two 3D short integer vectors `a` and `b`

### Lua Example
`local numberValue = vec3s_dot(a, b)`

### Parameters
| Field | Type |
| ----- | ---- |
| a | [Vec3s](structs.md#Vec3s) |
| b | [Vec3s](structs.md#Vec3s) |

### Returns
- `number`

### C Prototype
`f32 vec3s_dot(Vec3s a, Vec3s b);`
