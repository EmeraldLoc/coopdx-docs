
## [update_shell_speed](#update_shell_speed)

### Description
Controls Mario's speed when riding a Koopa Shell on the ground.

### Lua Example
`update_shell_speed(m)`

### Parameters
| Field | Type |
| ----- | ---- |
| m | [MarioState](structs.md#MarioState) |

### Returns
- None

### C Prototype
`void update_shell_speed(struct MarioState *m);`
