
## [le_set_light_pos](#le_set_light_pos)

### Description
Sets a lighting engine point light's position to `x`, `y`, `z`

### Lua Example
`le_set_light_pos(id, x, y, z)`

### Parameters
| Field | Type |
| ----- | ---- |
| id | `integer` |
| x | `number` |
| y | `number` |
| z | `number` |

### Returns
- None

### C Prototype
`void le_set_light_pos(s16 id, f32 x, f32 y, f32 z);`
