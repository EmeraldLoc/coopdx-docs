
## [djui_popup_create](#djui_popup_create)

### Description
Creates a popup that says `message` and has `lines`

### Lua Example
`djui_popup_create(message, lines)`

### Parameters
| Field | Type |
| ----- | ---- |
| message | `string` |
| lines | `integer` |

### Returns
- None

### C Prototype
`void djui_popup_create(const char* message, int lines);`
