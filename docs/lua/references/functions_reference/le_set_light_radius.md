
## [le_set_light_radius](#le_set_light_radius)

### Description
Sets a lighting engine point light's `radius`

### Lua Example
`le_set_light_radius(id, radius)`

### Parameters
| Field | Type |
| ----- | ---- |
| id | `integer` |
| radius | `number` |

### Returns
- None

### C Prototype
`void le_set_light_radius(s16 id, f32 radius);`
