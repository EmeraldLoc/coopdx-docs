
## [obj_set_cylboard](#obj_set_cylboard)

### Description
Enables cylindrical billboard rendering for an object

### Lua Example
`obj_set_cylboard(obj)`

### Parameters
| Field | Type |
| ----- | ---- |
| obj | [Object](structs.md#Object) |

### Returns
- None

### C Prototype
`void obj_set_cylboard(struct Object *obj);`
