
## [set_audio_fadeout](#set_audio_fadeout)

### Description
Sets the `fadeOutTime` of audio

### Lua Example
`set_audio_fadeout(fadeOutTime)`

### Parameters
| Field | Type |
| ----- | ---- |
| fadeOutTime | `integer` |

### Returns
- None

### C Prototype
`void set_audio_fadeout(u16 fadeOutTime);`
