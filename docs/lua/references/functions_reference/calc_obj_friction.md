
## [calc_obj_friction](#calc_obj_friction)

### Description
Determines an object's forward speed multiplier

### Lua Example
`local objFriction = calc_obj_friction(floor_nY)`

### Parameters
| Field | Type |
| ----- | ---- |
| floor_nY | `number` |

### Returns
- `number`

### C Prototype
`void calc_obj_friction(RET f32 *objFriction, f32 floor_nY);`
