
## [set_pause_menu_hidden](#set_pause_menu_hidden)

### Description
Sets if the pause menu elements are hidden, useful for creating custom pause menus

### Lua Example
`set_pause_menu_hidden(hidden)`

### Parameters
| Field | Type |
| ----- | ---- |
| hidden | `boolean` |

### Returns
- None

### C Prototype
`void set_pause_menu_hidden(bool hidden);`
