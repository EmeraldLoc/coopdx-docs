
## [get_level_course_num](#get_level_course_num)

### Description
Gets the level number's corresponding course number

### Lua Example
`local integerValue = get_level_course_num(levelNum)`

### Parameters
| Field | Type |
| ----- | ---- |
| levelNum | `integer` |

### Returns
- `integer`

### C Prototype
`s8 get_level_course_num(s16 levelNum);`
