
## [vec3s_length](#vec3s_length)

### Description
Calculates the length (magnitude) of the 3D short integer vector `a`

### Lua Example
`local numberValue = vec3s_length(a)`

### Parameters
| Field | Type |
| ----- | ---- |
| a | [Vec3s](structs.md#Vec3s) |

### Returns
- `number`

### C Prototype
`f32 vec3s_length(Vec3s a);`
