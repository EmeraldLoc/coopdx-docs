
## [le_light_exists](#le_light_exists)

### Description
Checks if a lighting engine point light corresponding to `id` exists

### Lua Example
`local booleanValue = le_light_exists(id)`

### Parameters
| Field | Type |
| ----- | ---- |
| id | `integer` |

### Returns
- `boolean`

### C Prototype
`bool le_light_exists(s16 id);`
