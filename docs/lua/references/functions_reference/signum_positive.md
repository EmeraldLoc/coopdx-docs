
## [signum_positive](#signum_positive)

### Description
Returns 1 for non-negative values and -1 for negative values

### Lua Example
`local integerValue = signum_positive(x)`

### Parameters
| Field | Type |
| ----- | ---- |
| x | `integer` |

### Returns
- `integer`

### C Prototype
`s32 signum_positive(s32 x);`
