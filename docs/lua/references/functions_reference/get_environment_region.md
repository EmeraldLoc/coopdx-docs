
## [get_environment_region](#get_environment_region)

### Description
Gets an environment region (gas/water boxes) height value

### Lua Example
`local integerValue = get_environment_region(index)`

### Parameters
| Field | Type |
| ----- | ---- |
| index | `integer` |

### Returns
- `integer`

### C Prototype
`s16 get_environment_region(u8 index);`
