
## [cur_obj_init_animation](#cur_obj_init_animation)

### Description
Initializes an animation for the current object by index

### Lua Example
`cur_obj_init_animation(animIndex)`

### Parameters
| Field | Type |
| ----- | ---- |
| animIndex | `integer` |

### Returns
- None

### C Prototype
`void cur_obj_init_animation(s32 animIndex);`
