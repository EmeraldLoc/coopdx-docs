
## [djui_chat_message_create](#djui_chat_message_create)

### Description
Creates a `message` in the game's chat box

### Lua Example
`djui_chat_message_create(message)`

### Parameters
| Field | Type |
| ----- | ---- |
| message | `string` |

### Returns
- None

### C Prototype
`void djui_chat_message_create(const char* message);`
