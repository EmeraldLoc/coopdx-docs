
## [djui_hud_render_line](#djui_hud_render_line)

### Description
Renders an DJUI HUD line onto the screen

### Lua Example
`djui_hud_render_line(p1X, p1Y, p2X, p2Y, size)`

### Parameters
| Field | Type |
| ----- | ---- |
| p1X | `number` |
| p1Y | `number` |
| p2X | `number` |
| p2Y | `number` |
| size | `number` |

### Returns
- None

### C Prototype
`void djui_hud_render_line(f32 p1X, f32 p1Y, f32 p2X, f32 p2Y, f32 size);`
