
## [gfx_resize](#gfx_resize)

### Description
Resizes a display list created by `gfx_create`

### Lua Example
`gfx_resize(gfx, newLength)`

### Parameters
| Field | Type |
| ----- | ---- |
| gfx | `Pointer` <`Gfx`> |
| newLength | `integer` |

### Returns
- None

### C Prototype
`void gfx_resize(Gfx *gfx, u32 newLength);`
