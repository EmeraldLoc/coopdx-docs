
## [smlua_text_utils_extra_text_get](#smlua_text_utils_extra_text_get)

### Lua Example
`local stringValue = smlua_text_utils_extra_text_get(index)`

### Parameters
| Field | Type |
| ----- | ---- |
| index | `integer` |

### Returns
- `string`

### C Prototype
`const char* smlua_text_utils_extra_text_get(s16 index);`
