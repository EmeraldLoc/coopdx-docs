
## [play_dialog_sound](#play_dialog_sound)

### Description
Plays a dialog sound corresponding to `dialogID`

### Lua Example
`play_dialog_sound(dialogID)`

### Parameters
| Field | Type |
| ----- | ---- |
| dialogID | `integer` |

### Returns
- None

### C Prototype
`void play_dialog_sound(s32 dialogID);`
