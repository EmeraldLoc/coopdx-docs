
## [play_cutscene](#play_cutscene)

### Description
Starts the execution of a predefined cutscene.
The camera transitions dynamically to follow the scripted sequence

### Lua Example
`play_cutscene(c)`

### Parameters
| Field | Type |
| ----- | ---- |
| c | [Camera](structs.md#Camera) |

### Returns
- None

### C Prototype
`void play_cutscene(struct Camera *c);`
