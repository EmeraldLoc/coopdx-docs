
## [le_set_ambient_color](#le_set_ambient_color)

### Description
Sets the lighting engine ambient color

### Lua Example
`le_set_ambient_color(r, g, b)`

### Parameters
| Field | Type |
| ----- | ---- |
| r | `integer` |
| g | `integer` |
| b | `integer` |

### Returns
- None

### C Prototype
`void le_set_ambient_color(u8 r, u8 g, u8 b);`
