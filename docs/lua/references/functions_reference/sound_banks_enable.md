
## [sound_banks_enable](#sound_banks_enable)

### Description
Disables `bankMask` soundbanks in `player`

### Lua Example
`sound_banks_enable(player, bankMask)`

### Parameters
| Field | Type |
| ----- | ---- |
| player | `integer` |
| bankMask | `integer` |

### Returns
- None

### C Prototype
`void sound_banks_enable(u8 player, u16 bankMask);`
