
## [vec3i_dot](#vec3i_dot)

### Description
Computes the dot product of the two 3D integer vectors `a` and `b`

### Lua Example
`local numberValue = vec3i_dot(a, b)`

### Parameters
| Field | Type |
| ----- | ---- |
| a | [Vec3i](structs.md#Vec3i) |
| b | [Vec3i](structs.md#Vec3i) |

### Returns
- `number`

### C Prototype
`f32 vec3i_dot(Vec3i a, Vec3i b);`
