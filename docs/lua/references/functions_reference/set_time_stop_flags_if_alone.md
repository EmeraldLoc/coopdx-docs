
## [set_time_stop_flags_if_alone](#set_time_stop_flags_if_alone)

### Description
Sets time stop flags only if the local player is alone

### Lua Example
`set_time_stop_flags_if_alone(flags)`

### Parameters
| Field | Type |
| ----- | ---- |
| flags | `integer` |

### Returns
- None

### C Prototype
`void set_time_stop_flags_if_alone(s32 flags);`
