
## [le_set_mode](#le_set_mode)

### Description
Sets the lighting engine mode to `mode`

### Lua Example
`le_set_mode(mode)`

### Parameters
| Field | Type |
| ----- | ---- |
| mode | [enum LEMode](constants.md#enum-LEMode) |

### Returns
- None

### C Prototype
`void le_set_mode(enum LEMode mode);`
