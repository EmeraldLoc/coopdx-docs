
## [djui_hud_get_raw_mouse_x](#djui_hud_get_raw_mouse_x)

### Description
Returns the x coordinate of the mouse relative to the screen

### Lua Example
`local numberValue = djui_hud_get_raw_mouse_x()`

### Parameters
- None

### Returns
- `number`

### C Prototype
`f32 djui_hud_get_raw_mouse_x(void);`
