
## [get_trajectory](#get_trajectory)

### Description
Gets a trajectory by `name`

### Lua Example
`local pointerValue = get_trajectory(name)`

### Parameters
| Field | Type |
| ----- | ---- |
| name | `string` |

### Returns
- `Pointer` <`Trajectory`>

### C Prototype
`Trajectory* get_trajectory(const char* name);`
