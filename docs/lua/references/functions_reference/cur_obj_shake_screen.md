
## [cur_obj_shake_screen](#cur_obj_shake_screen)

### Description
Shakes the camera around the current object with a given intensity

### Lua Example
`cur_obj_shake_screen(shake)`

### Parameters
| Field | Type |
| ----- | ---- |
| shake | `integer` |

### Returns
- None

### C Prototype
`void cur_obj_shake_screen(s32 shake);`
