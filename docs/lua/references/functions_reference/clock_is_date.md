
## [clock_is_date](#clock_is_date)

### Description
Checks whether it is the day given

### Lua Example
`local booleanValue = clock_is_date(month, day)`

### Parameters
| Field | Type |
| ----- | ---- |
| month | `integer` |
| day | `integer` |

### Returns
- `boolean`

### C Prototype
`bool clock_is_date(u8 month, u8 day);`
