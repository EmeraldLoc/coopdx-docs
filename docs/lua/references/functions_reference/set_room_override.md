
## [set_room_override](#set_room_override)

### Description
Overrides the current room Mario is in. Set to -1 to reset override

### Lua Example
`set_room_override(room)`

### Parameters
| Field | Type |
| ----- | ---- |
| room | `integer` |

### Returns
- None

### C Prototype
`void set_room_override(s16 room);`
