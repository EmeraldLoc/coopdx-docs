
## [get_lighting_color_ambient](#get_lighting_color_ambient)

### Description
Gets a value of the global ambient lighting color

### Lua Example
`local integerValue = get_lighting_color_ambient(index)`

### Parameters
| Field | Type |
| ----- | ---- |
| index | `integer` |

### Returns
- `integer`

### C Prototype
`u8 get_lighting_color_ambient(u8 index);`
