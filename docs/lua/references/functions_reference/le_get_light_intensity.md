
## [le_get_light_intensity](#le_get_light_intensity)

### Description
Gets a lighting engine point light's `intensity`

### Lua Example
`local numberValue = le_get_light_intensity(id)`

### Parameters
| Field | Type |
| ----- | ---- |
| id | `integer` |

### Returns
- `number`

### C Prototype
`f32 le_get_light_intensity(s16 id);`
