
## [djui_hud_set_filter](#djui_hud_set_filter)

### Lua Example
`djui_hud_set_filter(filterType)`

### Parameters
| Field | Type |
| ----- | ---- |
| filterType | [enum HudUtilsFilter](constants.md#enum-HudUtilsFilter) |

### Returns
- None

### C Prototype
`void djui_hud_set_filter(enum HudUtilsFilter filterType);`
