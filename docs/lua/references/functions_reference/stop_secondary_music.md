
## [stop_secondary_music](#stop_secondary_music)

### Description
Fades out secondary music over `fadeTimer`

### Lua Example
`stop_secondary_music(fadeTimer)`

### Parameters
| Field | Type |
| ----- | ---- |
| fadeTimer | `integer` |

### Returns
- None

### C Prototype
`void stop_secondary_music(u16 fadeTimer);`
