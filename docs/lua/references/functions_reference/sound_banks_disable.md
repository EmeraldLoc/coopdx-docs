
## [sound_banks_disable](#sound_banks_disable)

### Description
Enables `bankMask` soundbanks in `player`

### Lua Example
`sound_banks_disable(player, bankMask)`

### Parameters
| Field | Type |
| ----- | ---- |
| player | `integer` |
| bankMask | `integer` |

### Returns
- None

### C Prototype
`void sound_banks_disable(u8 player, u16 bankMask);`
