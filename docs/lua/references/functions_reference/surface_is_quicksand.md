
## [surface_is_quicksand](#surface_is_quicksand)

### Description
Checks if the surface is quicksand

### Lua Example
`local booleanValue = surface_is_quicksand(surf)`

### Parameters
| Field | Type |
| ----- | ---- |
| surf | [Surface](structs.md#Surface) |

### Returns
- `boolean`

### C Prototype
`bool surface_is_quicksand(struct Surface* surf);`
