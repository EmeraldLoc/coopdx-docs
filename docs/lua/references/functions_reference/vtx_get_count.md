
## [vtx_get_count](#vtx_get_count)

### Description
Gets the max count of vertices of a vertex buffer

### Lua Example
`local integerValue = vtx_get_count(vtx)`

### Parameters
| Field | Type |
| ----- | ---- |
| vtx | `Pointer` <`Vtx`> |

### Returns
- `integer`

### C Prototype
`u32 vtx_get_count(Vtx *vtx);`
