
## [obj_skip_interpolation](#obj_skip_interpolation)

### Description
Skips object interpolation for a frame

### Lua Example
`obj_skip_interpolation(o)`

### Parameters
| Field | Type |
| ----- | ---- |
| o | [Object](structs.md#Object) |

### Returns
- None

### C Prototype
`void obj_skip_interpolation(struct Object *o);`
