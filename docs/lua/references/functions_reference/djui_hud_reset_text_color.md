
## [djui_hud_reset_text_color](#djui_hud_reset_text_color)

### Description
Resets the current DJUI HUD text default color. This color is overridden by color codes

### Lua Example
`djui_hud_reset_text_color()`

### Parameters
- None

### Returns
- None

### C Prototype
`void djui_hud_reset_text_color(void);`
