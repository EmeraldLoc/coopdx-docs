
## [mario_pop_bubble](#mario_pop_bubble)

### Description
Makes Mario act like he was popped from a bubble. Useful for custom bubble popping behaviors.

### Lua Example
`mario_pop_bubble(m)`

### Parameters
| Field | Type |
| ----- | ---- |
| m | [MarioState](structs.md#MarioState) |

### Returns
- None

### C Prototype
`void mario_pop_bubble(struct MarioState* m);`
