
## [cur_obj_angle_to_home](#cur_obj_angle_to_home)

### Description
Returns the yaw angle from the current object toward its home position

### Lua Example
`local integerValue = cur_obj_angle_to_home()`

### Parameters
- None

### Returns
- `integer`

### C Prototype
`s16 cur_obj_angle_to_home(void);`
