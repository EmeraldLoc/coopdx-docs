
## [set_min_dialog_width](#set_min_dialog_width)

### Description
Dialog box customization: Sets the minimum width for a dialog box

### Lua Example
`set_min_dialog_width(width)`

### Parameters
| Field | Type |
| ----- | ---- |
| width | `integer` |

### Returns
- None

### C Prototype
`void set_min_dialog_width(s16 width);`
