
## [seq_player_fade_out](#seq_player_fade_out)

### Description
Fades out `player` with `fadeDuration`

### Lua Example
`seq_player_fade_out(player, fadeDuration)`

### Parameters
| Field | Type |
| ----- | ---- |
| player | `integer` |
| fadeDuration | `integer` |

### Returns
- None

### C Prototype
`void seq_player_fade_out(u8 player, u16 fadeDuration);`
