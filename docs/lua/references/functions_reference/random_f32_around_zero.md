
## [random_f32_around_zero](#random_f32_around_zero)

### Description
Returns a random floating-point value within +/- diameter/2

### Lua Example
`local numberValue = random_f32_around_zero(diameter)`

### Parameters
| Field | Type |
| ----- | ---- |
| diameter | `number` |

### Returns
- `number`

### C Prototype
`f32 random_f32_around_zero(f32 diameter);`
