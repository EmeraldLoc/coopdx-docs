
## [vec3i_length](#vec3i_length)

### Description
Calculates the length (magnitude) of the 3D integer vector `a`

### Lua Example
`local numberValue = vec3i_length(a)`

### Parameters
| Field | Type |
| ----- | ---- |
| a | [Vec3i](structs.md#Vec3i) |

### Returns
- `number`

### C Prototype
`f32 vec3i_length(Vec3i a);`
