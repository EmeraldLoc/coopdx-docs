
## [obj_mark_for_deletion](#obj_mark_for_deletion)

### Description
Marks an object to be unloaded at the end of the frame

### Lua Example
`obj_mark_for_deletion(obj)`

### Parameters
| Field | Type |
| ----- | ---- |
| obj | [Object](structs.md#Object) |

### Returns
- None

### C Prototype
`void obj_mark_for_deletion(struct Object *obj);`
