
## [djui_hud_get_mouse_scroll_y](#djui_hud_get_mouse_scroll_y)

### Description
Returns the amount scrolled vertically (-down/up+)

### Lua Example
`local numberValue = djui_hud_get_mouse_scroll_y()`

### Parameters
- None

### Returns
- `number`

### C Prototype
`f32 djui_hud_get_mouse_scroll_y(void);`
