
## [cur_obj_apply_drag_xz](#cur_obj_apply_drag_xz)

### Description
Applies drag to the current object's horizontal velocity components

### Lua Example
`cur_obj_apply_drag_xz(dragStrength)`

### Parameters
| Field | Type |
| ----- | ---- |
| dragStrength | `number` |

### Returns
- None

### C Prototype
`void cur_obj_apply_drag_xz(f32 dragStrength);`
