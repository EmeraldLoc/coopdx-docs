
## [set_sound_bank_override](#set_sound_bank_override)

### Description
Overrides the soundbank, set to -1 to reset

### Lua Example
`set_sound_bank_override(bank)`

### Parameters
| Field | Type |
| ----- | ---- |
| bank | `integer` |

### Returns
- None

### C Prototype
`void set_sound_bank_override(s32 bank);`
